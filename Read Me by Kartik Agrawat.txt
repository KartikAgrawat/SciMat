# Markdown Formula to MS Word Equation Converter 

### Developed by Kartik Agrawat

---

##  What Is This? (The Problem It Solves)

Have you ever copied a great response from an AI like Gemini or ChatGPT that contained mathematical formulas (Lambda, Gamma, fractions, etc.) or chemical formulas (like H2O, NH3, CO2, etc.), only to paste it into Microsoft Word and find it looking like a messy pile of code with dollar signs? 

For example, instead of a clean, professional equation, you get something ugly like this:
"The formula is $NH_3$ and the variance is $$\sigma_x^2$$"

Normally, you would have to spend hours manually re-typing every single symbol using Word's tedious equation editor. 

**This tool fixes that instantly.** It installs a permanent, custom **1-click button** directly at the top of your Microsoft Word window. When you paste text from an AI, you simply click your new button, and all that messy code instantly snaps into beautiful, professional, and fully editable Microsoft Word math equations or chemical formulas!

---

## How to Install It (1-Click Setup)

You don’t need to know anything about coding or system settings to get this running. 

1. Download this project folder to your computer.
2. Make sure Microsoft Word is closed.
3. Double-click the file named `setup.bat`.

That's it! A small command window will flash for a second to automatically place the files into Word's hidden startup folder. Once it finishes, the setup is permanently complete.

---

## How to Use It

Using the tool is incredibly simple:

1. Open **Microsoft Word** (any old file or a brand-new blank document).
2. Copy your text from Gemini or ChatGPT and **paste it normally** into Word.
3. Look at the very top left corner of your Word screen (above the standard "Home" ribbon toolbar). You will see your brand-new custom **Butterfly button** sitting there.
4. **Click the button once.**
5. Watch all the raw code and dollar signs instantly transform into clean, native mathematical equations or chemical formulas!

---

## What's Inside the Folder?

* `MarkdownConverter.dotm`: This is the master blueprint file. It holds the safe, automated background instructions and the visual layout for your new button.
* `setup.bat`: The automated installer engine. It takes the blueprint file and cleanly drops it into the exact folder Microsoft Word looks at when booting up. 

---

### System Compatibility
This tool works perfectly on all standard **Windows desktop versions of Microsoft Word (from Word 2010 all the way up to Word 2024 and Microsoft 365)**, but it will *not* run on Mac (Apple) computers, mobile apps, or the free web browser version of Word.
